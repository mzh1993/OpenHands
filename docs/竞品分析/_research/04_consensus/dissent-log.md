# 分歧与保留意见

## Manus

- 热度与订阅化成立，但公开、稳定、可独立验证的企业级客户证据不如 Replit、Dify、Lovable。
- 因此本轮不进入主样本，只作为候补观察。

## Bolt

- 企业版包装、微软渠道、设计系统叙事很强。
- 但公开商业证明密度仍偏少，容易被更强样本覆盖。

## Continue

- 在 AI code review / policy enforcement 方向有价值。
- 但和“自治软件工程 agent”不是同一强度层级，因此降为候补。

## Aider

- 开源影响力极强，是终端工作流的重要基线。
- 但公开 seat 经济、企业采用与产品化包装不足，不适合作为主报告核心样本。

## CodeGeeX

- 中国企业能力文档公开得很细，内网/私有化路径清楚。
- 但公开价格与公开客户规模数据较弱，外部可验证的商业信号不如 Trae / 通义灵码 / 文心快码。
